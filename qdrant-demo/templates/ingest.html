每行算一条独立记录，系统自动分配 ID。数据同时追加到 data/user_data.json 做本地备份。

或一键加载：

支持 JSON（[{"text":"..."}]）或 CSV（首行含 text 列）。当前上传只读取 text 字段；如需保留 category 等元数据，请使用下方命令行入库。

📌 数据存储说明
通过此页面写入的文本会：
  1. 被嵌入模型转换为向量，写入向量库（用于后续语义检索）
  2. 追加保存到项目根目录的 data/user_data.json（持久化备份；向量库数据清空后可用命令行重新入库）

💡 批量写入（命令行）
准备好格式为 [{"id": 1, "text": "...", "category": "..."}] 的 JSON 文件后执行：

python -m src.ingest /path/to/your_data.json   或   make ingest-large（载入 100 条大数据集）
